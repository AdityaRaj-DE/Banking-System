## How to Start
1. **Backend**:
    - Ensure MySQL is running on port 3307.
    - `npm install` and `node initDb.js` to setup the database.
    - `npm run dev` to start (Port 5000).
2. **Frontend**:
    - `npm install` in the `frontend` directory.
    - `npm run dev` to start (Port 5173).
3. **Login**: Use registered credentials or register a new account.
